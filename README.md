## Основні висновки

### Формула атестаційної оцінки:
```
А = (К + Е) × РПі × КРІ
```

де:
- **К** - класифікаційна оцінка (37 індикаторів)
- **Е** - експертна оцінка
- **РПі** - коефіцієнт регіональної підтримки (1,0–1,15)
- **КРІ** - коефіцієнт руйнувань інфраструктури (1,0–1,3)

### Розподіл внеску за блоками показників:

| Блок показників | Вага (Ki) | Внесок (%) |
|----------------|-----------|------------|
| Конкурсне фінансування проектів | 14,5 | 26,5% |
| Фінансова діяльність | 13,5 | 24,6% |
| Публікаційна активність | 11,3 | 20,6% |
| Інтелектуальна власність | 9,0 | 16,4% |
| Кадровий потенціал | 6,5 | 11,9% |

**Загальна сума вагових коефіцієнтів: 54,8**

### ТОП-5 найважливіших індикаторів:

1. Міжнародні патенти (7,3%)
2. Профінансовані міжнародні гранти (7,3% кожен, 2 індикатори)
3. Міжнародні пріоритетні програми - фінансування (5,5%)
4. Фінансування від іноземних замовників (5,5%)
5. Патенти на винаходи та сорти рослин (5,5%)

### Групи атестації:

- **Група А**: ≥ 75% (найвища)
- **Група Б**: 50–75% (висока)
- **Група В**: 25–50% (задовільна)
- **Група Г**: < 25% (не пройшли атестацію)
