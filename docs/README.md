# Веб-візуалізатор результатів атестації ЗВО України

Інтерактивна веб-платформа для аналізу та візуалізації результатів атестації закладів вищої освіти України.

## Особливості

- **Багатовимірна візуалізація**: Радарні діаграми, стовпчасті графіки, точкові діаграми та теплові карти
- **Порівняльний аналіз**: Можливість порівняння до 5 ЗВО одночасно
- **Автоматичні рекомендації**: Інтелектуальна система генерації рекомендацій на основі аналізу даних
- **Адаптивний дизайн**: Повна підтримка мобільних пристроїв та планшетів
- **Темна/світла тема**: Перемикання між темами оформлення
- **Експорт даних**: Можливість експорту результатів у PDF та Excel

## Технологічний стек

- **Frontend**: Vanilla JavaScript (ES6+)
- **Візуалізація**:
  - D3.js v7 (scatter plots, heatmaps)
  - Chart.js v4 (radar charts, bar charts)
- **Експорт**:
  - html2pdf.js (PDF export)
  - SheetJS (Excel export)
- **Стилізація**: CSS Grid, Flexbox, CSS Custom Properties

## Використання

### Локальний запуск

1. Переконайтеся, що всі JSON файли знаходяться у директорії `data/`:
   - all_results.json
   - methodology.json
   - stats_by_direction.json
   - validation.json

2. Запустіть локальний веб-сервер:
   ```bash
   # Python 3
   python -m http.server 8000

   # або Node.js
   npx serve
   ```

3. Відкрийте браузер та перейдіть до `http://localhost:8000`

### Публікація на GitHub Pages

1. Переконайтеся, що всі файли знаходяться в директорії `gh-pages/`

2. У налаштуваннях репозиторію GitHub:
   - Settings → Pages
   - Source: Deploy from a branch
   - Branch: main → /gh-pages → Save

3. Сайт буде доступний за адресою:
   ```
   https://[username].github.io/[repository]/
   ```

## Функціонал

### 1. Вибір ЗВО для аналізу

- Основний ЗВО для детального аналізу
- За замовчуванням: "Криворізький державний педагогічний університет"
- Можливість вибору наукового напряму (1-7)

### 2. Порівняльний аналіз

- Вибір 2-5 ЗВО для порівняння
- Паралельне відображення на всіх графіках
- Виділення різними кольорами

### 3. Візуалізації

#### Радарна діаграма (Radar Chart)
- Візуалізація 5 блоків індикаторів:
  - Кадровий потенціал
  - Фінансова діяльність
  - Публікаційна активність
  - Інтелектуальна власність
  - Конкурсне фінансування

#### Стовпчаста діаграма (Bar Chart)
- Відображення всіх 37 індикаторів
- Можливість порівняння з іншими ЗВО
- Інтерактивні підказки з вагами індикаторів

#### Точкова діаграма (Scatter Plot)
- Позиція ЗВО відносно медіанних значень
- Візуалізація сильних та слабких сторін
- Розмір точок відповідає вазі індикатора

#### Теплова карта (Heatmap)
- Топ-10 ЗВО за обраним напрямом
- Кольорова шкала інтенсивності показників
- Можливість порівняння між закладами

### 4. Детальна таблиця

- Всі 37 індикаторів з деталями
- Нормовані значення
- Ваги та внески у загальну оцінку

### 5. Автоматичні рекомендації

- Аналіз слабких показників
- Пріоритизація рекомендацій
- Конкретні дії для покращення
- Групові рекомендації залежно від атестаційної групи

### 6. Експорт даних

- **PDF**: Повний звіт з усіма візуалізаціями
- **Excel**: Таблиця з показниками для подальшого аналізу

## Налаштування

### Зміна ЗВО за замовчуванням

У файлі `js/app.js` змініть значення:

```javascript
this.currentInstitution = 'Назва вашого ЗВО';
```

### Налаштування кольорової схеми

У файлі `css/main.css` у розділі `:root` можна змінити кольори:

```css
--primary: #2563eb;
--secondary: #64748b;
/* ... інші кольори */
```

### Адаптація методології

Файл `data/methodology.json` містить:
- Ваги індикаторів
- Структуру блоків
- Назви наукових напрямів
- Групи атестації

## Розробка

### Додавання нової візуалізації

1. Створіть метод у `js/visualizations.js` або `js/charts.js`
2. Додайте контейнер в `index.html`
3. Викличте метод у `js/app.js` в функції `createVisualizations()`

### Додавання нової рекомендації

У файлі `js/recommendations.js` в методі `createRecommendation()` додайте:

```javascript
'Блок': {
    'I##': {
        title: 'Заголовок',
        description: 'Опис',
        actions: ['Дія 1', 'Дія 2']
    }
}
```
