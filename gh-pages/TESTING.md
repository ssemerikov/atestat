# Інструкція з тестування веб-візуалізатора

## Швидкий старт

### 1. Локальний запуск

```bash
cd /home/cc/claude_code/atestat/gh-pages
python3 -m http.server 8000
```

Відкрийте браузер: `http://localhost:8000`

### 2. Перевірка завантаження даних

Відкрийте консоль браузера (F12) та перевірте:

```javascript
// Перевірка завантаження JSON
fetch('./data/all_results.json')
    .then(r => r.json())
    .then(d => console.log('Results loaded:', d.length, 'records'))
    .catch(e => console.error('Error:', e));

fetch('./data/methodology.json')
    .then(r => r.json())
    .then(d => console.log('Methodology loaded:', d))
    .catch(e => console.error('Error:', e));
```

## Тестові сценарії

### Сценарій 1: Базовий аналіз

1. Відкрити сторінку
2. Перевірити, що обрано "Криворізький державний педагогічний університет"
3. Залишити напрям "Всі напрями"
4. Натиснути "Аналізувати"
5. Перевірити, що всі візуалізації з'явились
6. Перевірити інформацію про заклад
7. Прокрутити до рекомендацій

**Очікуваний результат:** Всі дані відображаються коректно

### Сценарій 2: Порівняння ЗВО

1. Обрати основний ЗВО
2. Обрати 3 ЗВО для порівняння
3. Натиснути "Аналізувати"
4. Перевірити радарну діаграму (4 ЗВО)
5. Перевірити стовпчасту діаграму (4 ЗВО)
6. Перевірити легенду з кольорами

**Очікуваний результат:** Всі 4 ЗВО відображаються різними кольорами

### Сценарій 3: Фільтрація за напрямом

1. Обрати ЗВО
2. Обрати науковий напрям (наприклад, "Природничо-математичний")

4. Перевірити, що даті відповідають напряму
5. Перевірити топ-10 в хeatmap

**Очікуваний результат:** Дані фільтруються за напрямом

### Сценарій 4: Експорт даних

1. Проаналізувати ЗВО
2. Натиснути "Експорт в PDF"
3. Дочекатися завантаження PDF

6. Відкрити Excel файл

**Очікуваний результат:** Обидва файли завантажуються та відкриваються

1. Перевірити початкову тему
2. Натиснути кнопку перемикання теми
3. Перевірити, що всі елементи перефарбувались

## Автоматичне тестування (опціонально)

Створіть файл `test.html`:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Tests</title>
</head>
<body>
    <script type="module">
        import DataLoader from './js/data-loader.js';

        async function runTests() {
            console.log('Starting tests...');

            const loader = new DataLoader();

            try {
                const data = await loader.loadAll();
                console.log('✅ Data loaded successfully');

                const institutions = loader.getInstitutions(data);
                console.log('✅ Institutions:', institutions.length);

                const defaultInst = institutions.find(i =>
                    i.includes('Криворізький'));
                console.log('✅ Default institution found:', defaultInst);

                console.log('All tests passed! ✅');
            } catch (error) {
                console.error('❌ Test failed:', error);
            }
        }

        runTests();
    </script>
</body>
</html>
```
